# Fitness Tracker App

Flutterで作成されたフィットネストラッカーアプリです。
食事管理、トレーニング記録、進捗トラッキング（画像比較機能付き）を提供します。

## セットアップ手順

このプロジェクトは手動でファイルを作成したため、以下の手順で実行環境を整えてください。

1. Flutter SDKがインストールされていることを確認してください。
2. ターミナルでプロジェクトルート（`pubspec.yaml`がある場所）に移動します。
3. 以下のコマンドを実行して、プラットフォーム固有のファイルを生成します。
   ```bash
   flutter create .
   ```
   ※ `android` や `ios` フォルダが生成されます。

4. 依存関係をインストールします。
   ```bash
   flutter pub get
   ```

5. アプリを実行します。
   ```bash
   flutter run
   ```

## 機能概要

### 1. 食事管理
- **目標設定**: カロリー、PFCバランス（タンパク質、脂質、炭水化物）の目標を設定できます。
- **食事記録**: 食品名と栄養素を入力して記録します。
- **サマリー**: 今日の摂取カロリーとPFCバランスをグラフで確認できます。

### 2. トレーニング記録
- **記録**: 種目名、重量、回数、セット数、インターバル、メモを記録できます。
- **履歴**: 過去のトレーニング履歴を一覧で確認できます。
- **前回比較**: 新しい記録を入力する際、同じ種目名の過去の記録（重量・回数など）が表示されます。

### 3. 進捗トラッキング
- **身体測定**: 体重、腹囲、体脂肪率を記録し、グラフで推移を確認できます。
- **画像記録**: 体型の写真を撮影・保存できます。
- **画像比較**: 最新の画像と過去の画像を半透過で重ねて表示し、体型の変化を視覚的に比較できます。

## 注意事項
- データはローカルデータベース（SQLite）に保存されます。アプリを削除するとデータも消えます。
- iOSシミュレーターでカメラ機能を使用する場合、`Info.plist` にカメラ使用の権限設定が必要になる場合があります（`flutter create` で生成されるデフォルト設定で足りない場合）。
